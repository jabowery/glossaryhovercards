Glossary with Interactive Hovercards

This page demonstrates the glossary hovercard system. Terms marked with **bold asterisks** are interactive.

Example Definitions

Action

An *action* is any transformation $A$ acting on a **Structure Table** or **Count Table** that:

*GA interpretation:* An action is a linear operator $A$ on multivectors such that: projectors ($A^2 = A$) implement selection, linking, hiding; involutions ($A^2 = 1$) implement polarity and time symmetry; unitaries ($A^\dagger A = 1$) implement reversible evolution; signed-additive operators respect cancellation of **Empty Parts**.

All other entries below are defined as patterns or uses of actions.

Thing

A *thing* is anything that is transformed non-trivially by some **Action**. Formally: $t$ is a thing iff there exists an action $A$ with $A(t) \neq A(0)$ under the induced **Relative Identity**. GA: any multivector not annihilated by all relevant projectors.

Discriminator

A *discriminator* $x$ is a distinguished action $A_x$ whose effect defines **Relative Identity**. Two things are identical-for-$x$ if $A_x$ transforms them in the same way. GA: a chosen family of projectors, involutions, or unitaries encoding the perspective of $x$.

Structure Table

A table containing structural information that can be transformed by an **Action**.

Count Table

A table containing count data that can be operated on by various **Action** operators.

Empty Parts

Components that cancel out under signed-additive operators in the context of an **Action**.

Relative Identity

An equivalence relation defined by a **Discriminator** $x$. Two **Thing**s are considered identical relative to $x$ if they are transformed identically by the discriminator's **Action**.
